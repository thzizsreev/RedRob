# Tools

CLI utilities for validation, reporting, and ad-hoc retrieval tests.

| Script | Purpose |
|--------|---------|
| [`validate_submission.py`](validate_submission.py) | Validate submission CSV format |
| [`audit_top100_honeypots.py`](audit_top100_honeypots.py) | Pre-submit honeypot rate audit (Stage 3 threshold) |
| [`build_team_view.py`](build_team_view.py) | Stage 5 → HTML ranking page (`outputs/team_views/`) |
| [`build_eliminations_view.py`](build_eliminations_view.py) | Stages 2–5 elimination funnel (`outputs/eliminations/`) |
| [`build_reasoning_lookup.py`](build_reasoning_lookup.py) | Build reasoning lookup from CSV |
| [`rank.py`](rank.py) | Fast CPU retrieval smoke test |
| [`eliminations_view/`](eliminations_view/) | Eliminations HTML generator package |
